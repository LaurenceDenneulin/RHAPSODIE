## Installation

In the package manager:

```julia
pkg> add https://github.com/LaurenceDenneulin/Rhapsodie.jl
```
if you use HTTPS or:

```julia
pkg> add git@github.com:LaurenceDenneulin/Rhapsodie.jl
```
if you use SSH.
